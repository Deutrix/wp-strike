WP Strike plugin for showing image or text instead website content.

== Description ==

Style and schedule options included. Bots and crawlers will see the content, but the visitors will see the strike message.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-strike` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings page and set the strike message and schedule
